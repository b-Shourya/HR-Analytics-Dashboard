# HR Analytics Dashboard

## Overview

This project is an interactive HR Analytics Dashboard developed in Power BI to analyze employee data and present meaningful business insights. It provides a clear view of workforce distribution, salary trends, gender diversity, and departmental performance through interactive visualizations and key performance indicators (KPIs).

The dashboard is designed to help HR teams and management understand workforce data and support informed decision-making.



## Objectives

- Analyze employee demographics and workforce distribution.
- Compare employee count across different departments.
- Understand gender distribution within the organization.
- Examine salary trends and employee compensation.
- Present important HR metrics using KPIs.
- Enable users to explore the data through interactive filters and slicers.
